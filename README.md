# Cloud Computing Labs ☁️

Репозиторий содержит лабораторные работы по курсу **облачных вычислений**.  
Каждая лабораторная работа реализована в отдельной ветке.

---

## 📌 Структура репозитория

- `lab2` — Jenkins + сборка проекта на OpenStack
- `lab3` — Infrastructure as Code (Heat + Jenkins)
- `lab4` — Деплой через Jenkins
- `lab5_6` — Terraform + Ansible. Полный CI/CD Pipeline
- `lab7` — Kubernetes-деплой

---

## Lab 1 — OpenStack и CLI

Создание виртуального сервера вручную в OpenStack и 
настройка OpenStack CLI.  
Продемонстрировано создание и удаление виртуальных машин 
через командную строку.

---

## Lab 2 — Jenkins CI

Подключение собственного сервера как Jenkins Node.  
Создан Jenkins job для Python-проекта с настройкой сборки 
на отдельном агенте и архивацией артефактов в DockerHub.

---

## Lab 3 — Infrastructure as Code (Heat)

Реализован YAML-шаблон Heat для автоматического создания 
инфраструктуры в OpenStack.  
Развертывание инфраструктуры перенесено в Jenkins.

---

## Lab 4 — Деплой через Jenkins

Настроен автоматический деплой собранного артефакта 
на созданную инфраструктуру.  
Реализована передача артефакта и перезапуск приложения 
через Jenkins job.

---

## Lab 5 — Terraform + Ansible

Инфраструктура создаётся с помощью Terraform.  
Настройка системного и прикладного ПО выполняется через Ansible.

---

## Lab 6 — Полный CI/CD Pipeline

Объединены этапы сборки, создания инфраструктуры и деплоя в единый Jenkins Pipeline.  
Реализован автоматизированный процесс доставки приложения от кода до развертывания.

---

## Lab 7 — Kubernetes

Удалено создание инфраструктуры на уровне VM.  
Реализован деплой приложения в Kubernetes-кластер с доставкой артефакта в контейнеризированное окружение.


---

## 🎯 Цель проекта

Освоение полного цикла DevOps:  
от ручного развертывания инфраструктуры до автоматизированного CI/CD и Kubernetes-деплоя.
